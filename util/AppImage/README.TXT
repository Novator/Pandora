
===How to build Pandora.AppImage===

1) Download standart files:
   - AppRun-x86_64
   - appimagetool-x86_64.AppImage
2) Put "AppRun-x86_64" to "Pandora.AppMount" subdir with name "AppRun"
   and set it executable
3) Fresh "Pandora.AppMount/usr/pandora" subdir with new Pandora files
4) Run "Build.Pandora.AppImage.sh" script to build Pandora.AppImage
